## *ABOUT GAME*
This is a short platformer made in gdscript on the godot version. The game will take about 5 minutes.

![](https://pin.it/4o9NqkA)
You can do a few simple things, such as:
* walk, run and jump
* collect coins
* use ladders, buttons and doors
* shoot fireballs
* kill enemies
* etc
